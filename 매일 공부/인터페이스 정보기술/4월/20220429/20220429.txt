웹 브라우저 디버깅.
Resume( F8 ) : 스크립트 실행을 다시 시작함.
Step( F9 ) : 다음 문을 실행합니다.
Step over( F10 ) : 다음 명령어를 실행하되, 함수 안으로 들어가진 않음.
Step into( F11 ) : 현재 실행 중인 함수의 실행을 계속 이어가다가 함수 본문 마지막 줄에서 실행을 멈춥니다.
