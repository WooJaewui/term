 # 용어 정리.

SAP System Application Product.
비즈니스 운영과 고객 지원을 위한 ERP 소프트웨어를 개발하는 소프트웨어 기업이다.

WSDL Web Services Description Language.
특정서비스의 설명과 접근방법 등을 XML 언어기반으로 제공하는 기술이다.

UDDI Universal Description, Descovery Integration.
자신들의 서비스내용을 인터넷상에 등록할 수 있게하는 XML 기반의 등록처이다.

SOAP Simple Object Access Protocol.
다른 언어로 다른 플랫폼에서 빌드된 애플리케이션이 통신할 수 있도록 설계된 최초의 표준 프로토콜.
프로토콜이기 때문에 복잡성과 오버헤드를 증가시키는 빌트인 룰을 적용하므로 로드 시간이 길 수 있다.

REST Representational State Transfer.
웹 서비스와 모바일 애플리케이션 경량화의 필요에 맞춤 아키텍처 원칙 세트이다.

MIS Management Information System.
기업 경영에서 의사결정의 유효성을 높이기 위하여 기업 관련 데이터를 관리하는 시스템.

ERP Enterpries Resource Planning.
기업 내의 자원을 효율적으로 관리하는 통합정보시스템.

DML Data Mainpulation Language.
데이터 조작 언어.



 # 연동가이드.

연동 유형.
WEB 연동 : HTTP를 이용한 JSON, HTML, XML 서비스 제공 서버.
DB 연동 : SQL, PROCEDURE 제공하는 서비스 서버.
SAP 연동 : SAP에서 제공하는 JCO 라이브러리를 사용하여 RFC 호출.
WSDL 연동 : WDSL은 SOAP 서비스를 제공하는 서버.

개발 프로세스.
개발 시작 전 연동 양식과 인터페이스는 정의되어 있어야 한다.

연동 API 생성.
일반 연동 API : 연동 양식 결재문서 기안화면에서 호출할 연동 API.
상태 연동 API : 연동 양식으로 기안된 문서가 결재 프로세스가 진행되면서 상태 호출할 연동 API.
